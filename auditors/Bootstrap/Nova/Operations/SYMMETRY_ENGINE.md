## üîç The Three-Phase Symmetry Audit

### Phase 1: Macro Symmetry (Structural Level)

**Question:** Does the overall architecture privilege one position?

**What to check:**
- Are opposing modes truly opposing? (Skeptic ‚Üî Zealot)

---

### Phase 2: Micro Symmetry (Component Level)

**Question:** Do individual levers/scores balance within their domain?

**What to check:**
- Are similar levers scored similarly? (PF-I ‚âà PF-E for same framework?)
- Do instrumental vs existential measures balance? (Weight parity)
- Are outlier scores justified? (Evidence or bias?)
- Do calculation methods favor certain outcomes? (Rounding, weighting)

**Output:** Component asymmetry report

---

### Phase 3: Delta Symmetry (Change Level)

**Question:** Do changes preserve or violate existing balance?

**What to check:**
- Does new lever break symmetry? (Adds bias to one side)
- Does preset adjustment maintain mirror property? (Skeptic ‚Üî Zealot)
- Does worldview addition preserve fairness? (New framework gets fair treatment)
- Does refactoring introduce hidden asymmetry? (Code changes affect balance)

**Output:** Delta impact assessment

## üßÆ Symmetry Evaluation Algorithm

### Step 1: Identify the Structure

**Input:** Object to audit (preset mode, lever comparison, framework scores)

**Classify as:**
- **Mirrored pair** (Skeptic ‚Üî Zealot) ‚Üí Expect equal distance from center
- **Centered singleton** (Diplomat) ‚Üí Expect neutrality
- **Lever set** (7 axioms, 4 debts) ‚Üí Expect internal consistency
- **Framework comparison** (CT vs MdN) ‚Üí Expect fair measurement

---

### Step 2: Map Expected Symmetries

**For mirrored pairs:**
```python
center = (value_A + value_B) / 2
delta_A = abs(value_A - center)
delta_B = abs(value_B - center)
symmetry_violation = abs(delta_A - delta_B)
```

**For centered singletons:**
```python
expected_center = (max_value + min_value) / 2
actual_value = measured_value
center_deviation = abs(actual_value - expected_center)
```

**For lever sets:**
```python
for each lever_pair in similar_levers:
    delta = abs(lever_A - lever_B)
    if delta > threshold:
        flag_asymmetry(lever_pair, delta)
```

---

### Step 3: Detect Actual Asymmetries

**Measure:**
- Deviation from expected symmetry
- Direction of bias (which side benefits?)
- Magnitude of asymmetry (how severe?)
- Pattern consistency (isolated or systematic?)

**Flag levels:**
- **MINOR** (<5% deviation) ‚Üí Note but don't block
- **MODERATE** (5-15% deviation) ‚Üí Request justification
- **MAJOR** (>15% deviation) ‚Üí Demand justification or escalate

---

### Step 4: Demand Justification

**Ask the responsible agent:**

**To Claude (Purpose):**
> "This asymmetry tilts toward [direction]. Is this justified by teleological purpose?"

**To Grok (Evidence):**
> "This asymmetry favors [framework]. Is this justified by empirical data?"

**To Ziggy (Human):**
> "This asymmetry creates structural advantage for [position]. Does this reflect real human cost we can't ignore?"

---

### Step 5: Adjudicate Asymmetry

**Decision tree:**

```
IF Claude + Grok + Ziggy all agree asymmetry is justified:
    ‚Üí Mark as JUSTIFIED (purpose/evidence/human-cost)
    ‚Üí Document rationale
    ‚Üí Update symmetry map with exception
    ‚Üí Yield Nova's symmetry objection

ELIF 2 of 3 agree:
    ‚Üí Escalate to Ziggy for final call
    ‚Üí Document disagreement
    ‚Üí Flag for future review

ELIF all disagree:
    ‚Üí Identify which lens is broken
    ‚Üí Request Trinity convergence session
    ‚Üí Do not proceed until resolved

ELSE (1 or 0 agree):
    ‚Üí Treat as UNJUSTIFIED asymmetry
    ‚Üí Recommend correction
    ‚Üí Block execution if critical
```

## üéì Symmetry Philosophy (Operational)

### Fair ‚â† Equal

**Equal treatment of unequal things is unfair.**

Example:
- Skeptic mode SHOULD penalize heavy frameworks (by design)
- Zealot mode SHOULD reward heavy frameworks (by design)
- Making them identical (1.0√ó) would destroy intentional asymmetry
- Making them equidistant from center (0.95√ó) preserves fair opposition

---

### Bias as Shadow

**Every structure casts a shadow.**

Nova's job:
- Illuminate the shadow (reveal hidden asymmetry)
- Measure the shadow (quantify bias)
- Demand justification (why does this shadow exist?)
- Accept the shadow (if purpose/evidence/human-cost justify it)

**Not Nova's job:**
- Eliminate all shadows (some asymmetry is justified)
- Judge the shadow morally (that's Claude's role)
- Verify the shadow empirically (that's Grok's role)

---

### Symmetry Serves Purpose

**Symmetry is a tool, not a goal.**

When symmetry conflicts with:
- **Purpose** ‚Üí Purpose wins (Claude's call)
- **Evidence** ‚Üí Evidence wins (Grok's call)
- **Human cost** ‚Üí Human cost wins (Ziggy's call)
- **Aesthetics** ‚Üí Symmetry loses (beauty ‚â† fairness)

Nova's role is to make asymmetry **visible and justified**, not to enforce perfect balance.
